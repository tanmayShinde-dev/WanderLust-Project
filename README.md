# 🌍 WanderLust – Travel Listing Web Application

WanderLust is a full-stack web application that allows users to explore, create, and manage travel/property listings. The platform includes user authentication, CRUD operations, and cloud-based image handling.

---

## 🚀 Features

- 🔐 User Authentication (Login / Signup)
- 📝 Create, Read, Update, Delete (CRUD) listings
- 🖼️ Image upload and storage using Cloudinary
- ⭐ Review and rating system
- 🛡️ Schema validation for data integrity
- 📱 Responsive UI for better user experience

---

## 🛠️ Tech Stack

- **Frontend:** EJS, Bootstrap, CSS
- **Backend:** Node.js, Express.js
- **Database:** MongoDB Atlas
- **Cloud Storage:** Cloudinary
- **Deployment:** Render

---

## 📂 Project Structure


controllers/ → Business logic
models/ → Database schemas
routes/ → Application routes
views/ → EJS templates
public/ → Static assets (CSS, JS)
utils/ → Helper functions

app.js → Entry point
